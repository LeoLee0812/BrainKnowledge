# 鸡尾酒会效应——听到自己名字突然醒

## 定义
> [!note] 定义
> 在吵闹环境里你以为屏蔽掉了背景对话，可一旦其中出现自己的名字，注意力会被瞬间夺走。这就是==鸡尾酒会效应==——名字不是"被你听到"，而是从后台**自下而上突围**到意识里。

## 神经机制 / 原理

**反直觉前提**：过滤不发生在"听"的入口，而发生在"要不要让你意识到"的出口。大脑其实**一直在全量处理所有声音**，包括你自以为没在听的那条。

1. **全量解析（==颞上回== 听觉皮层）**：即使完全没注意，被忽略的声音流仍被做了**语义级别**的解码——大脑知道那串音节是不是"你的名字"。所以筛选发生在语义之后，而非声音一进来就被砍。

2. **注意闸门（==背外侧前额叶== + 顶叶）**：自上而下的注意控制系统平时把"光标"锁在你正聊的人身上，靠 ==GABA== 能抑制回路主动**压低**其他声音流。你"听不见"旁边桌，不是没处理，是被压住了。

3. **警报点火（==前岛叶== + ==前扣带回== 组成的突显网络）**：名字是被强化过几十万次的**超高突显刺激**。后台解析一旦撞中"名字"模板，突显网络瞬间点火，**强行把注意光标从当前对话拽走**。

4. **整体唤醒（==蓝斑核== 释放 ==去甲肾上腺素== + 多巴胺上调）**：这一拽把整体唤醒度拉高，就是你主观感到的"突然一激灵、醒了"。

**一句话串起来**：全量解析（颞上回）→ 撞中高突显模板 → 突显网络拉警报（岛叶/ACC）→ 抢走前额叶的注意闸门 → 蓝斑核拉高唤醒。你不是"开始听"，而是"被名字劫持了"。

## 现象 / 例子

- 约 1/3 的人，在被忽略的耳道里塞入自己名字会被检测到。
- **越焦虑、工作记忆容量越小**的人越容易被劫持——前额叶的闸门关不紧。
- 困倦或喝醉时唤醒系统迟钝，叫别人名字反而更难"激活"对方。
