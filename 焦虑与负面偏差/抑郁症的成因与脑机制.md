# 抑郁症的成因与脑机制

## 定义

> [!note] 定义
> 抑郁症（Major Depressive Disorder）是一种涉及神经递质失衡、HPA轴过度激活、神经炎症和脑结构重塑的多因素疾病，而非简单的"心情不好"。

## 神经机制 / 原理

### 神经递质失衡

三种关键递质与抑郁密切相关：

- ==血清素（5-HT）==：调节情绪稳定和睡眠，信号传递减弱时出现持续低落、焦虑、失眠
- ==去甲肾上腺素（NE）==：管注意力和动力，不足时表现为疲惫、精神涣散、反应迟钝
- ==多巴胺（DA）==：奖赏回路的核心燃料，活性降低导致"快感缺失"——以前喜欢的事全都没意思了

> [!warning] 化学失衡不是全部
> 传统的"化学失衡"理论已被认为过于简化。如果只是递质不够，SSRI 应该吃下去几小时就起效——但实际需要数周，说明背后有更深层的机制。

### 慢性压力 → HPA轴 → 海马萎缩

这是从压力演变为抑郁的核心通路：

1. 长期压力让 ==下丘脑-垂体-肾上腺轴（HPA轴）== 持续亢奋
2. 大量 ==皮质醇==（压力激素）持续释放
3. 皮质醇具有神经毒性，长期高水平使 ==海马体== 的神经元萎缩、==BDNF（脑源性神经营养因子）== 表达下降、新生神经元减少
4. 海马萎缩后，对 HPA 轴的负反馈抑制减弱 → 皮质醇更高 → 恶性循环

### 神经炎症

慢性压力、感染、不良生活方式会激活大脑中的 ==小胶质细胞==（大脑的免疫细胞），释放炎症因子（IL-6、TNF-α）：

- 干扰血清素和多巴胺的合成通路
- 直接损伤神经元
- 抑制海马区的神经再生

研究发现大多数抑郁症患者血液和脑脊液中炎症因子水平明显偏高。

### 脑区层面

- ==前额叶皮层==（理性调控）活性降低 → 无法有效抑制负面情绪
- ==杏仁核==（恐惧和负面情绪中心）过度活跃 → 持续焦虑和恐惧感
- ==默认模式网络（DMN）== 过度活跃 → 停不下来的反刍思维

### 遗传因素

抑郁症有 40%~50% 的发病风险来自基因（5-HTTLPR、BDNF 基因等），但基因决定的是"易感性"，环境压力才是扣下扳机的手。

## 现象 / 例子

一个人长期加班、睡眠不足（慢性压力）→ 皮质醇持续偏高 → 海马神经元受损、小胶质细胞激活释放炎症因子 → 血清素合成被干扰、多巴胺奖赏回路钝化 → 开始对什么都提不起兴趣、持续低落、脑子里反复想"我是不是很没用" → 前额叶无力刹车、杏仁核持续报警 → 临床抑郁。

这不是"想太多"，是大脑在多个层面同时出了问题。
