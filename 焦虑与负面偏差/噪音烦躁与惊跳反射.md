# 噪音烦躁与惊跳反射

## 定义
> [!note] 定义
> 噪音烦躁与惊跳反射是两种不同时间尺度的听觉应激反应：持续噪音通过慢性皮质醇积累导致烦躁，突发噪音则在毫秒内触发脑干惊跳反射并引发肾上腺素激增。

---

## 现象一：持续噪音为什么让人烦躁

### 神经机制

**通路：听觉皮层 → 杏仁核 → HPA轴 → 皮质醇慢性积累**

1. **听觉皮层无法完全适应不规则噪音**  
   大脑对稳定信号有"适应/抑制"机制（神经元放电率下降），但对==频率和音量不稳定的噪音==（如施工声、聊天声），适应机制失效——每次变化都被当作新刺激重新处理。

2. **前扣带回（ACC）的冲突检测**  
   前扣带回持续检测"我想要安静 vs 实际上有噪音"这个冲突，产生==注意力资源争夺==，占用前额叶带宽（与杂乱环境的视觉噪声机制相同）。

3. **杏仁核低水平持续激活**  
   听觉输入通过丘脑直接投射到杏仁核（"低路"通路），持续噪音 = 杏仁核持续微量激活 → ==HPA轴被慢性激活== → 皮质醇水平抬高。

4. **皮质醇蓄积效应**  
   皮质醇升高的副产品是：易激怒、注意力涣散、对进一步刺激的阈值降低（更容易烦）——形成正反馈：越烦 → 对噪音越敏感 → 越烦。

## 现象二：突然噪音为什么让心脏扑通

### 神经机制

**两条并行通路，速度差异极大：**

#### 通路一：脑干惊跳回路（极速，8–20ms）
```
突发声音 → 耳蜗核 → 脑桥网状核（PnC）→ 脊髓运动神经元
```
- 这条路==完全绕过皮层==，不需要"意识到"就已经做出反应
- 结果：肌肉瞬间收缩（肩膀耸起、眼睛闭合）、心率骤然加速
- 这是进化保留下来的原始防御机制，对所有哺乳动物都一样

#### 通路二：杏仁核威胁评估（较慢，100–300ms）
```
突发声音 → 丘脑 → 杏仁核 → 下丘脑 → 交感神经系统 → 肾上腺髓质
```
- 杏仁核评估"这个声音是威胁吗？" → 即便判断为误报，==肾上腺素已经释放==
- 肾上腺素效应：心跳加速（心悸感）、血压上升、瞳孔扩大、肌肉紧张
- ==蓝斑核==同时大量释放去甲肾上腺素 → 全脑急剧唤醒，完全清醒

#### 为什么睡眠中更难受
- 睡眠时基础唤醒水平极低（尤其深睡期皮质醇最低），突发噪音造成的==唤醒落差极大==
- 没有"预期"可以提前调低杏仁核敏感度（**预脉冲抑制**，prepulse inhibition），无预警 = 惊跳幅度最大
- REM期杏仁核本身更活跃，情绪反应被放大

### 为什么感觉"很久才能平静下来"
肾上腺素的半衰期约 2–3 分钟，但其引发的==交感神经激活==需要更长时间回落；同时皮质醇在此后 15–30 分钟内持续升高，让人保持高度警觉、难以重新入睡。

---

## 应用实践与启发

- **睡前降低惊跳敏感度**：白噪音/粉噪音作为"声学背景"，让突发噪音的相对响度下降，减小唤醒落差
- **耳塞 vs 白噪音**：完全静音反而让微小声音显得更突出（对比更大），白噪音掩蔽更有效
- **惊跳后的平复**：慢速腹式呼吸（呼气长于吸气）激活迷走神经，加速肾上腺素代谢、降心率
- **持续噪音烦躁**：前扣带回资源被占满时，切换任务比"忍"更有效；戴降噪耳机直接切断输入
