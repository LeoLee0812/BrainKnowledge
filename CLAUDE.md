# 我的知识库（BrainKnowledge）

关于元认知、认知科学、大脑工作原理的系统性笔记库。每个文件对应一个知识点。

## 关于我

- 身份：独立开发者 / 创作者，对神经科学/脑科学感兴趣的人
- 关注领域：认知科学、神经科学、元认知、心理学、学习方法
- 偏好：用中文、不要过度格式化、解释要讲清神经机制而非泛泛而谈

## Vault 结构

这是一个**纯本地 Obsidian vault**——没有网页构建、没有社交媒体脚本、不推送任何远程仓库，就是本地的一堆 Markdown 文件。

当前为**按主题分文件夹**——笔记按 `分类` property 归到 7 个主题文件夹下，文件夹名 = `分类` 取值。

- 根目录只放 3 个文件：`index.md`、`README.md`、`CLAUDE.md`
- 7 个主题文件夹：`前额叶与自控/`、`睡眠与晨起/`、`多巴胺与奖赏/`、`记忆/`、`社交与情绪/`、`焦虑与负面偏差/`、`学习与认知效率/`
- 每篇笔记放进与其 `分类` property 一致的文件夹，按「知识点中文名.md」命名（如 `前额叶与自控/前额叶如何抑制冲动.md`）
- `index.md`：**唯一的分类索引（MOC）**，按 7 大主题归类全部笔记，是检索入口
- `README.md`：知识库简介，目录指向 `index.md`
- `.obsidian/`：Obsidian 配置，不要手动改
- `.claude/skills/`：本库内置的 Obsidian skills（见「写作风格」）

> 注意：Obsidian 的 `[[wikilink]]` 按文件名在全库解析，与所在文件夹无关，所以移动文件不会断链；新建笔记只需放进正确的主题文件夹即可。

笔记多由 Claude Code 的 `/元认知` 命令自动生成与维护。

## 笔记格式

**使用 Obsidian Flavored Markdown**，每篇笔记顶部必须有 YAML frontmatter properties，正文用 callout / wikilink / highlight 等 Obsidian 语法：

```markdown
---
date: YYYY-MM-DD
分类: <7 大主题之一，见下>
tags:
  - <主题关键词，注意不要含斜杠，否则会被当成嵌套标签>
神经递质:
  - <如 多巴胺 / 血清素 / 皮质醇，没有可省略>
脑区:
  - <如 前额叶皮层 / 杏仁核 / 海马，没有可省略>
---

# 知识点标题

## 定义
> [!note] 定义
> 一句话能说清的先说清。

## 神经机制 / 原理
（涉及的脑区、神经递质、回路；讲机制不讲结论。关键术语用 `==高亮==`）

## 现象 / 例子
（结合具体场景帮助理解）

> [!info] 相关笔记
> - [[相关笔记A]]
> - [[相关笔记B]]

## 参考来源
（论文、书籍或可信资料）
```

要点：
- `分类`、`神经递质`、`脑区` 三个 property 是为 `.base` 视图筛选准备的，新笔记务必填好。
- `tags` 里的值**不要带斜杠**（`/` 会被 Obsidian 解析成嵌套标签层级）。
- 各章节标题可按知识点灵活调整，但「frontmatter + 标题 + 相关笔记 + 参考来源」必须保留。

## 行为规则

- **可以**：新增笔记、补充内容、创建 `[[双向链接]]`、生成摘要、整理分类、更新 `index.md` 索引
- **不可以**：删除已有笔记内容、修改已有笔记的核心论述、改动我的原始记录
- 新建笔记必须遵循上面的格式，命名用知识点中文名
- 相关知识点之间用 `[[双向链接]]` 互相关联（如失眠相关的笔记互链）
- 每次新增或整理后，把新笔记归入 `index.md` 对应主题，保持可检索

## 写作风格（Obsidian skills）

本库 `.claude/skills/` 内置了几个 Obsidian skills，写笔记时**优先调用**：

- `obsidian-markdown`：wikilink、callout、properties、highlight 等 Obsidian 语法——新建/重写笔记时先调它确认写法。
- `obsidian-bases`：把 properties 渲染成可筛选的 `.base` 表格/卡片视图。
- `json-canvas`：把知识点连成 `.canvas` 关系图。
- `obsidian-cli` / `defuddle`：命令行操控 Obsidian、把网页正文转干净 Markdown 作为参考素材。

## 主题分类（`分类` property 的取值，便于归类与互链）

新笔记的 `分类` 必须是下列 7 项之一：

- **前额叶与自控**：冲动抑制、目标劫持、前额叶疲劳、注意力
- **睡眠与晨起**：睡眠机制、晨起情绪、皮质醇节律、REM、做梦
- **多巴胺与奖赏**：预测误差、游戏成瘾、心流、奖赏回路
- **记忆**：遗忘曲线、具身编码、既视感、童年失忆
- **社交与情绪**：血清素、群体认同、社会比较、情绪波动
- **焦虑与负面偏差**：先想坏结果、焦虑逃避循环、压力激素
- **学习与认知效率**：反馈延迟、习得性无助、智力、元认知
